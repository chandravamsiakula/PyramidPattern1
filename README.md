# Pyramid Pattern Problem

## 📌 Problem Description
Given multiple integers as input, for each integer **n**, print a pyramid pattern using `*` symbols with **n** rows.  
After each pattern, print a separator line.

---

## 📥 Input
The first line contains the number of test cases.  
The next lines contain integers representing the height of each pyramid.

## input
3

4

5

3
